## What is this?

Live an entire lifetime — from birth to death — in a world driven by a **local large language model**. No cloud API. No subscription. No data leaves your machine.

Every event, every choice, every consequence is uniquely generated by AI. No two playthroughs are ever the same.

## Features

### Core Gameplay
- **Three Built-in Worlds** — Modern City, Immortal Cultivation (Xianxia), Medieval Warring States
- **AI-Generated Narrative** — Every event, choice, and outcome is uniquely written by the LLM
- **DND-Style Dice Checks** — Stat + dice roll vs. difficulty, with dramatic animated reveals
- **Boss Battles** — Critical turning points where you invest stat points for massive rewards or devastating consequences
- **Seven Core Stats + HP** — Constitution, Intelligence, Charisma, Wealth, Luck, Social, Willpower
- **Fate Bond System** — Your choices create bonds that echo across your entire lifetime
- **Epic Endings** — AI-written epitaphs that weave your life story into poetic prose

### Technical
- **100% Local & Private** — All AI processing runs on your computer via Ollama
- **Choose Your AI Model** — Built-in model manager lets you download, switch, and compare models
- **Mod Support** — Create any world via JSON files — Harry Potter, Cyberpunk, your favorite anime
- **Async Pipeline** — Three-phase architecture (dice → narrative → next event) for seamless gameplay
- **Cross-Platform** — Windows, Linux, macOS

### Model Management
- **First-Launch Setup** — Guides new players through choosing and downloading their first AI model
- **In-Game Model Switcher** — Change models anytime from the Settings menu without restarting
- **Recommended Models Catalog** — Curated list with VRAM requirements, speed ratings, and quality tiers
- **Background Downloads** — Download new models while browsing the menu, with real-time progress

## Quick Start

### Prerequisites

| Software | Required | Download |
|----------|----------|----------|
| **Python** | 3.10+ | [python.org](https://www.python.org/downloads/) |
| **Ollama** | Latest | [ollama.com](https://ollama.com/download) |
| **Godot** | 4.4 (to build from source) | [godotengine.org](https://godotengine.org/) |

### Option A: Play from Release (Recommended)

1. Download the latest release from [Releases](../../releases) or [itch.io](https://YOUR_USERNAME.itch.io/infinite-life-simulator)
2. Run the installer:
   - **Windows:** Double-click `install.bat`
   - **Linux/macOS:** `chmod +x install.sh && ./install.sh`
3. Launch the game:
   - **Windows:** Double-click `launcher.bat`
   - **Linux/macOS:** `./launcher.sh`
4. On first launch, the game will guide you through downloading an AI model

### Option B: Build from Source

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/infinite-life-simulator.git
cd infinite-life-simulator

# Install backend dependencies
cd backend
pip install -r requirements.txt
cd ..

# Start the backend
cd backend && python main.py &

# Open the Godot project
# Open Godot 4.4 → Import → navigate to the project folder → Open
# Press F5 to run
```

## Choosing an AI Model

The game includes a built-in model manager. On first launch, you'll be prompted to download a model. You can also change models anytime in **Settings → AI Model**.

| Model | VRAM | Speed | Quality | Best For |
|-------|------|-------|---------|----------|
| `llama3.2:3b` | ~3 GB | ⚡ Fast | ★★★☆☆ Good | Low-end hardware, quick testing |
| **`llama3.1:8b`** | **~6 GB** | **🔄 Medium** | **★★★★☆ Great** | **Default — best balance** |
| `qwen2.5:7b` | ~6 GB | 🔄 Medium | ★★★★☆ Great | Multilingual worlds, non-English mods |
| `mistral:7b` | ~6 GB | 🔄 Medium | ★★★★☆ Great | Creative writing variety |
| `llama3.1:70b-q4` | ~40 GB | 🐌 Slow | ★★★★★ Exceptional | High-end hardware only |

**To use a custom model** not in the catalog, pull it manually via Ollama CLI:
```bash
ollama pull your-model:tag
```
Then switch to it in Settings — any model installed in Ollama will appear.

### Three-Phase Async Pipeline

The core gameplay loop eliminates perceived waiting time:

1. **Player makes a choice** → `POST /roll_dice` returns instantly with dice data
2. **Dice animation plays (~10s)** → Backend LLM generates narrative in background
3. **Narrative displays (~8s)** → Backend LLM generates next event in background
4. **Next event appears seamlessly** → No loading screen needed

This means the player **never waits** for the AI — the ~18 seconds of animation perfectly covers both LLM calls.

## Modding

Create your own worlds with a single JSON file. See **[MODDING_GUIDE.md](MODDING_GUIDE.md)** for the complete reference.

### Quick Example

```json
{
  "info": {
    "name": "Wizarding World",
    "rules": "Magic requires wands. The Statute of Secrecy hides wizards from Muggles.",
    "description": "The world of witches and wizards."
  },
  "attributes": {
    "display_names": {
      "constitution": "Vitality",
      "intelligence": "Magical Talent",
      "wealth": "Galleons"
    }
  },
  "character": { "start_age": 11, "max_age": 120 },
  "node_ages": [11, 14, 17],
  "node_templates": {
    "11": "The Sorting Hat sits upon your head..."
  }
}
```

Save as `backend/world_data/worlds/your_world.json` → restart → it appears in the world selector.

## Deployment

### For itch.io

Since this game requires a local Python backend + Ollama, it cannot run as an HTML5 game in the browser. Deploy as a **downloadable** package:

1. **Export from Godot** for each platform (Windows, Linux, macOS)
2. **Create a ZIP** per platform containing:
   - The exported game executable
   - The entire `backend/` directory
   - `install.bat`/`install.sh`, `launcher.bat`/`launcher.sh`
   - `README.md`, `MODDING_GUIDE.md`, `LICENSE`
3. **Upload to itch.io:**
   - Go to [itch.io Dashboard](https://itch.io/dashboard) → **Create new project**
   - **Title:** Infinite Life Simulator
   - **Classification:** Game
   - **Kind of project:** Downloadable
   - **Upload** each platform ZIP, marking the appropriate OS
   - **Cover image:** 630×500px (315:250 ratio)
   - **Tags:** `simulation`, `ai`, `life-sim`, `godot`, `local-ai`, `text-based`, `rpg`
   - **Description:** Use the itch.io editor for a formatted version of this README
   - Set **Pricing** to free (or donation)
   - Set **Visibility** to Public when ready

### For GitHub

1. Create repository: `github.com/YOUR_USERNAME/infinite-life-simulator`
2. Push all source code (both `backend/` and `godot_project/`)
3. Create releases with pre-built binaries for each platform
4. Use GitHub Actions for automated builds (optional — see `.github/workflows/` examples)

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "Backend not connected" | Make sure `launcher.bat`/`launcher.sh` started the backend, or run `cd backend && python main.py` manually |
| "Ollama not found" | Install from [ollama.com](https://ollama.com/download), then run `install.bat`/`install.sh` |
| Model download stuck at 0% | Check internet connection. Ollama downloads from `registry.ollama.ai` |
| Very slow generation | Try a smaller model (`llama3.2:3b`) in Settings → AI Model |
| Game window blank | Wait for "Server is ready!" in the launcher terminal before opening the game |
| Events contain wrong language | Make sure the world config matches the model's strengths. `llama3.1:8b` is best for English |

## 🎮 Gameplay & Key Features

At its core, this is a text-based RPG driven by a dynamic LLM game master. 

**How to Play:**
You begin by creating a character, distributing initial stat points (Constitution, Intelligence, Charisma, Wealth, Luck, Social, Willpower), and selecting a universe. Every "year" of your life, the AI generates a unique event tailored to your current age, stats, and the world's lore. You choose how to react, and the outcome is decided by a D20-style dice roll modified by your attributes.

**Why it stands out:**
* **Infinite Replayability:** There are no hardcoded dialogue trees. The LLM dynamically crafts every narrative, meaning no two lives will ever be the same.
* **Epic Tribulations (Boss Battles):** At critical milestones in your life, you will face massive, life-or-death crises. You must permanently sacrifice your hard-earned stats to survive these judgments of fate.
* **Seamless Immersion:** Enjoy a polished UI with cinematic typewriter effects, screen shakes, and dynamic soundscapes that elevate the text adventure into a gripping visual novel experience.
* **Hybrid AI Power:** Run completely offline with local models (via Ollama) for zero latency and absolute privacy, or seamlessly plug in your favorite cloud APIs (DeepSeek, OpenAI, etc.) for top-tier reasoning.

## 🌟 Open-Source Vision & Philosophy

I open-sourced this project with a simple hope: **I want interested players and developers to feel completely free to modify, break, expand, and reshape this game.**

My original intention was to harness the power of Large Language Models to generate deeply unpredictable and wildly diverse life journeys. The true magic of this architecture lies in its flexibility. By simply tweaking the world configuration files, you have the absolute freedom to build your own custom universes. 

With the right prompt constraints, you can recreate the worlds of your favorite novels, movies, or anime—allowing you to cross paths with familiar characters and experience entirely new storylines written just for you. 

I hope this framework empowers every player to craft, explore, and ultimately find a life they are truly satisfied with. Feel free to fork, mod, and let your imagination run wild!
